Erklärung und Entstehungshintergrund:


Qr-Codes (Quick Response Code) ist ein zweidimensionaler Code welcher aus einer quadratischen Matrix mit schwarzen und weißen Quadraten/Punkten besteht.
Diese Matrix wird Module genannt und stellt die kodierten binären Daten dar.
Die Bekannten drei Makierungen in den vier äusseren Ecken, geben die Oriertierung vor.
Erstmalig wurde der Qr-Code 1994 von der Tochterfirma Denso Wave entwickelt, welche von ihrem Mutterkonzern Toyota beauftragt wurde,
um die Markierung von Baugruppen und Komponenten für die Logistik des Automobilriesen zu vereinfachen.

Standards:


Es gibt mehr Qr-Code standards, der Unterschied liegt meist in der Codierung und der visuelen Darstellung.
