Fehlerkorrekur in RAID

RAID ("redundant array of independent disks") ist ein System zum Erstellen von Computer Speichern bestehend aus mehreren Speichermedien(Festplatten, SSDs, ...).
Da beim Speichern von Daten in großem Maß, wie es bei RAID-Systemen meist der Fall ist, die Sicherheit dieser sehr wichtig ist, sind im RAID-Standart Fehlerkorrekurmöglichkeiten integriert, die hauptsächlich gut funktionieren, wenn ganze Festplatten ausfallen sollten. In den verschiedenen RAID-Versionen, die man auch "RAID level" nennt, unterscheiden sich neben der Art, wie die Daten auf die verschiedenen Speichermedien verteilt werden auch in ihrer Fehlerkorrekur.

RAID 0

RAID 0 verteilt jediglich die Daten gleichmäßig auf mehrere Speichermedien, bietet jedoch keine Möglichkeit zur Korrektur von entstandenen Fehlern in den Daten.

RAID 1

Bei RAID 1 werden mindestes zwei Festplatten benötigt. Jeder Festplatte wird eine Weitere als sogenannte Spiegelfestplatte zugewiesen. Werden Daten gespeichert, so werden sie auf Beiden gespeichet.
Sollte nun eine der Festplatten versagen, so kann der Controller des RAID-Systems weiterhin die Daten von der Spiegelfestplatte lesen. Um dann weiterhin eine Schutzkopie zu haben, muss die kaputte Festplatte ersetzt oder eine neue Spiegelfestplatte zugewiesen werden.

RAID 2

Die Daten werden bei RAID 2 auf Bit-Ebene, anstatt auf Datenblock-Ebene, gleichmäßig über Speichermedien. zur Fehlerkorrektur wird in die Daten ein Hamming-Code verwendet.

RAID 5

RAID 5 benötigt mindestes 3 Festplatten, kann aber auch mit bis zu 16 arbeiten. In einem Prozess namens "Striping" werden die Nutzdaten in etwa gleichgroße Blöcke aufgeteilt und auf die Festplatten verteilt. Auf einer weiteren Festplatte, auf der keiner der Blöcke gespeichert wird, werden Paritätsdaten gespeichert. Diese entstehen durch ein exklusives Oder (XOR, $) der anderen Blöcke. Werden verschiedene Dateien nacheinander gespeichet, so werden die Paritätsdaten im Kreis rotierend auf einer anderen Festplatte gespeichert. Dies soll verhindern, dass, wenn eine eigens dafür ausgewählte Festplatte, beschädigt wird alle Paritätsdaten darauf erneuert werden müssen.

Weitere RAID-level

Es gibt noch weitere RAID-level: 3, 4, 6 und 7, sowie 10, welches aus den Prinzipien von RAID 0 und RAID 1 zusammengesetzt ist. Dennoch ist RAID 5 die am meisten verwendete RAID-Art.
